Simple repository to run quantum teleportation on real quantum devices using the Qiskit library.

- The file `qubit_teleportation.ipynb` implements the qubit teleportation protocol on a real quantum device.
- The file `qudit_2^n_teleportation.ipynb` implements qudit teleportation by mapping the qudit to qubits. Therefore, it is only suitable for dimensions that are powers of 2.
