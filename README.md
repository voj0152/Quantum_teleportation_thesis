Repository to run quantum teleportation on real quantum devices using the Qiskit library.

- The file `qubit_teleportation.ipynb` implements the qubit teleportation protocol on a real quantum device, also Mthree mitigation.
- The file `qudit_2^n_teleportation.ipynb` implements qudit teleportation by mapping the qudit to qubits. Therefore, it is only suitable for dimensions that are powers of 2, also Mthree mitigation.
- In folder `results` are stored results of each experiment and qiskit circuit representation.

The Qiskit documentation was used as a reference for creating these notebooks. Specifically the notebook [quantum-teleportation.ipynb](https://github.com/Qiskit/documentation/blob/be3df1bd249279a6883a0c28601571f2abe38fab/learning/modules/computer-science/quantum-teleportation.ipynb),
